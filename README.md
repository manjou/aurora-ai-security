## 🎯 Project Overview

Complete enterprise network design and security infrastructure implementation for Aurora AI Security GmbH, demonstrating:

- Network architecture & VLAN segmentation
- SIEM deployment (Wazuh) & security monitoring
- Firewall configuration (pfSense) & access control
- Compliance framework implementation (GDPR, NIS2, IT-SiG, FADP)
- SOC workflow design & incident response
- Penetration testing & security validation
